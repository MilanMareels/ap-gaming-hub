# AP Gaming Hub

Een modern platform voor de AP Hogeschool gaming community. Studenten kunnen hier gaming gear (PC's, PS5's, Switch) reserveren, teams vinden, en op de hoogte blijven van events. Beheerders hebben een uitgebreid dashboard om alles te managen.

## Features

### Voor Studenten

- **Gear Reserveren:** Boek eenvoudig een High-End PC, PS5 of Nintendo Switch.
- **Agenda & Events:** Bekijk direct de openingsuren en geplande evenementen.
- **Highscores:** Check de leaderboards en zie wie de beste scores heeft.
- **Gaming Teams:** Vind informatie over actieve teams en rosters.
- **Info & Community:** Huisregels, FAQ en directe link naar de Discord server.

### Voor Admins (Dashboard)

- **Reservatie Beheer:** Bekijk alle boekingen, filter op datum en verwijder indien nodig.
- **Inventory Management:** Beheer het aantal beschikbare consoles en controllers live.
- **Agenda & Events:** Plan events in die automatisch zichtbaar worden in de weekplanning.
- **Openingsuren:** Pas de openingsuren per dag aan (Open/Gesloten/Event).
- **Highscores & Rosters:** Beheer leaderboards en teamlijsten.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Taal:** TypeScript
- **Styling:** Tailwind CSS
- **Database & Auth:** Firebase (Firestore & Authentication)
- **Icons:** Lucide React

## Getting Started

### 1. Clone de repository

```bash
git clone https://github.com/milanmareels/ap-gaming-hub.git
cd ap-gaming-hub
```

### 2. Installeer dependencies

```bash
npm run dev
# of
npm install
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.
